generator

может понадобиться сделать chmod 777 и исполняемый файл

В общем запускается из консоли, у меня это

python3 generator.py

просит количество тетриминок, которые будем крафтить,
главное чтоб названия начинались с

/fillit_check*
./fillit_obrazech*

в файл difference2.txt записывает ПЕРВЫЕ 10 комбинаций на которых были
расхождения в выводе, в файл count_fails сколько вообще ошибок было,
работает не быстро, поэтому ждать до конца не обязательно, можешь
абортнуть исполнение и посмотреть что уже успело в файлы записаться

и походу из за разницы операционных систем может придётся перекомпилить
исполняемые файлы и переназвать чтоб заработало корректно
